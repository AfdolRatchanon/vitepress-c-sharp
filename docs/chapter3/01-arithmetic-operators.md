# Arithmetic Operators (ตัวดำเนินการทางคณิตศาสตร์)

> 💡 **เป้าหมาย:** เข้าใจการคำนวณคณิตศาสตร์ใน C# ทั้งแบบพื้นฐาน (บวก ลบ คูณ หาร) และแบบพิเศษที่โปรแกรมเมอร์ใช้บ่อย (หารเอาเศษ, การเพิ่ม/ลดค่าทีละหนึ่ง)

## 📖 ทฤษฎีและแนวคิด (Theory & Concepts)

### 1. Basic Arithmetic (คณิตศาสตร์พื้นฐาน)

C# รองรับเครื่องหมายคณิตศาสตร์พื้นฐานเหมือนเครื่องคิดเลข:
- `+` (บวก)
- `-` (ลบ)
- `*` (คูณ)
- `/` (หาร)

```text
ข้อควรระวังเรื่องการหาร (Division):
ใน C# การหารจำนวนเต็ม (int) กับจำนวนเต็ม (int) จะได้ผลลัพธ์เป็น "จำนวนเต็ม" เท่านั้น (ปัดเศษทิ้งเสมอ!)

  int a = 10;
  int b = 3;
  Console.WriteLine(a / b);  // ผลลัพธ์คือ 3 (ไม่ใช่ 3.333)

ถ้าต้องการทศนิยม ตัวตั้งหรือตัวหารต้องเป็นทศนิยม (double/float/decimal) อย่างน้อย 1 ตัว:
  double c = 10.0;
  int d = 3;
  Console.WriteLine(c / d);  // ผลลัพธ์คือ 3.33333333333333
```

### 2. Modulo Operator (หารเอาเศษ: `%`)

เครื่องหมาย `%` (อ่านว่า โมดูโล หรือ มอด) คือ **การหารเพื่อเอาเฉพาะเศษ** ไม่ใช่เปอร์เซ็นต์!
นี่คือเครื่องหมายที่โปรแกรมเมอร์ใช้บ่อยมากที่สุดตัวหนึ่ง

```text
  10 / 3  = 3 เศษ 1  (ดังนั้น 10 % 3 ผลลัพธ์คือ 1)
  15 / 4  = 3 เศษ 3  (ดังนั้น 15 % 4 ผลลัพธ์คือ 3)
  10 / 2  = 5 เศษ 0  (ดังนั้น 10 % 2 ผลลัพธ์คือ 0)

ประโยชน์ของ Modulo:
1. หาเลขคู่/เลขคี่: ถ้า `x % 2 == 0` แปลว่าเป็นเลขคู่
2. จำกัดขอบเขต: เช่น หมุนรอบ (Clock arithmetic) เวลา 25:00 น. คือ `25 % 24 = 1` (ตีหนึ่ง)
```

### 3. Increment & Decrement (เพิ่มและลดค่าทีละ 1)

การเพิ่มหรือลดค่าทีละ 1 เป็นสิ่งที่เจอบ่อยมาก เช่น การนับคะแนน C# จึงมีเครื่องหมายทางลัด:
- `++` (เพิ่มค่าขึ้น 1)
- `--` (ลดค่าลง 1)

**ตำแหน่งมีความหมาย (Prefix vs Postfix):**
```text
  int x = 5;

  // 1. Postfix (ทำทีหลัง): ใช้ค่าเก่าก่อน แล้วค่อยบวก
  int y = x++;   // y จะได้ค่า 5 ส่วน x จะกลายเป็น 6

  // 2. Prefix (ทำก่อน): บวกก่อน แล้วค่อยเอาค่าไปใช้
  int z = ++x;   // x จะถูกบวกเป็น 7 ทันที แล้ว z จะได้ค่า 7
```

### 4. Assignment Operators (ตัวดำเนินการกำหนดค่าแบบย่อ)

แทนที่จะเขียน `score = score + 10;` C# ให้คุณเขียนย่อได้:
- `+=` (บวกเพิ่ม)  -> `x += 5;` (เหมือน `x = x + 5;`)
- `-=` (ลบออก)    -> `x -= 5;`
- `*=` (คูณเพิ่ม)  -> `x *= 5;`
- `/=` (หารออก)   -> `x /= 5;`
- `%=` (มอดเอาเศษ) -> `x %= 5;`

## 💻 ตัวอย่างโค้ด (Code Implementation)

### ตัวอย่างที่ 1: การใช้ Modulo เช็คเลขคู่/คี่ และคำนวณเวลา

::: code-group
```csharp [Program.cs]
Console.WriteLine("=== ระบบคำนวณทางคณิตศาสตร์ ===");

int totalMinutes = 135; // สมมติเรามีเวลา 135 นาที

// ต้องการแปลงเป็น ชั่วโมง:นาที
int hours = totalMinutes / 60;    // หารเอาส่วนเต็ม (135/60 = 2)
int minutes = totalMinutes % 60;  // หารเอาเศษ (135%60 = 15)

Console.WriteLine("เวลาทั้งหมด " + totalMinutes + " นาที");
Console.WriteLine("เท่ากับ " + hours + " ชั่วโมง " + minutes + " นาที");

// ตัวอย่างการเช็คเลขคู่คี่เบื้องต้น
int number = 42;
int remainder = number % 2;
Console.WriteLine("\nเลข " + number + " หารด้วย 2 เหลือเศษ " + remainder);
Console.WriteLine("(ถ้าเศษเป็น 0 คือเลขคู่, เศษเป็น 1 คือเลขคี่)");
```
:::

**Expected Output:**
```
=== ระบบคำนวณทางคณิตศาสตร์ ===
เวลาทั้งหมด 135 นาที
เท่ากับ 2 ชั่วโมง 15 นาที

เลข 42 หารด้วย 2 เหลือเศษ 0
(ถ้าเศษเป็น 0 คือเลขคู่, เศษเป็น 1 คือเลขคี่)
```

### ตัวอย่างที่ 2: Postfix vs Prefix อธิบายแบบเห็นภาพ

::: code-group
```csharp [Program.cs]
int a = 10;
Console.WriteLine("ค่าเริ่มต้น a = " + a);

// Postfix: เอา a ไปให้ WriteLine พิมพ์ก่อน พิมพ์เสร็จ a ค่อยบวก 1
Console.WriteLine("สั่ง a++ (Postfix): " + a++); 
Console.WriteLine("ค่า a ตอนนี้: " + a); // a กลายเป็น 11 แล้ว

Console.WriteLine("----------------");

int b = 10;
Console.WriteLine("ค่าเริ่มต้น b = " + b);

// Prefix: บวก 1 ให้ b ทันที (กลายเป็น 11) แล้วค่อยส่งให้ WriteLine พิมพ์
Console.WriteLine("สั่ง ++b (Prefix): " + ++b); 
Console.WriteLine("ค่า b ตอนนี้: " + b); // b ก็เป็น 11
```
:::

**Expected Output:**
```
ค่าเริ่มต้น a = 10
สั่ง a++ (Postfix): 10
ค่า a ตอนนี้: 11
----------------
ค่าเริ่มต้น b = 10
สั่ง ++b (Prefix): 11
ค่า b ตอนนี้: 11
```

  *(Hint: ใช้ `/` สำหรับหาจำนวนแบงก์ และ `%` สำหรับหาเศษเงินที่เหลือไปทอนแบงก์ถัดไป)*

- **โจทย์ที่ 2:** เดาผลลัพธ์ของโค้ดนี้โดยไม่ต้องรันโปรแกรม:
  ```csharp
  int x = 5;
  x += 3;
  x *= 2;
  x--;
  Console.WriteLine(x);
  ```

::: details 💡 คำใบ้ (Hint)
- โจทย์ 1: 
  `เงินที่เหลือ = 857`

  `เงินที่เหลือ = 857 % 100` (เหลือ 57)
  ทำแบบนี้ไปเรื่อยๆ กับแบงก์ 50 และเหรียญ 10
- โจทย์ 2: ค่อยๆ ไล่ค่า `x` ทีละบรรทัด 5 -> 8 -> 16 -> 15

## 🔥 Challenge (โจทย์ท้าทาย!)

- **โจทย์:** คุณมีตัวเลข `int n = 12345;` เขียนโค้ดเพื่อสกัดเอา **เฉพาะตัวเลขหลักร้อย** (นั่นคือเลข 3) ออกมาใส่ตัวแปรใหม่ให้ได้ โดยใช้แค่เครื่องหมาย `/` และ `%` เท่านั้น! ห้ามแปลงตัวเลขเป็นข้อความ (`string`) เด็ดขาด

## 🗣️ ทบทวน (Review)

::: details ❓ คำถามทบทวนความเข้าใจ

**คำถาม 1:** เครื่องหมาย `%` (Modulo) ใช้ทำอะไร และบอกประโยชน์มาอย่างน้อย 1 ข้อ
**แนวคำตอบ:** ใช้สำหรับการหารเพื่อเอา "เศษ" เท่านั้น ประโยชน์ที่เจอบ่อยคือ ใช้ตรวจสอบว่าเลขนั้นเป็นเลขคู่หรือเลขคี่ (ถ้านำมา `% 2` แล้วได้ `0` คือเลขคู่ ถ้าได้ `1` คือเลขคี่)

**คำถาม 2:** โค้ด `int a = 10; int b = 3; double result = a / b;` ค่าของ `result` จะเป็นเท่าไหร่? ทำไมไม่เป็น `3.33`?
**แนวคำตอบ:** เป็น `3` (ในรูปแบบ `3.0` บนหน่วยความจำ) สาเหตุเพราะ `a` และ `b` เป็น `int` ทั้งคู่ การหารระหว่าง `int` กับ `int` ใน C# จะทำการตัดเศษทิ้งก่อนเสมอ ทำให้ได้ผลลัพธ์เป็น `3` จากนั้นจึงค่อยนำเลข 3 ไปเก็บในกล่อง `double` ทีหลัง หากอยากได้ `3.33` ต้องมีตัวใดตัวหนึ่งเป็นทศนิยมก่อน เช่น `double result = (double)a / b;`
:::
