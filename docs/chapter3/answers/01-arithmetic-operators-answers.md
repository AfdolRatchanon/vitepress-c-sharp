# เฉลย: Arithmetic Operators

## 🎯 เฉลย Mini Exercise

**โจทย์ที่ 1:** โปรแกรมตู้ทอนเงิน (ทอนแบงก์ 100, 50, และเหรียญ 10)
**เฉลย:**
เป้าหมายของโจทย์นี้คือฝึกการใช้หารตัดเศษ `/` ร่วมกับโมดูโล `%` ซึ่งเป็นหัวใจสำคัญของอัลกอริทึมประเภทจัดการธนบัตร

```csharp
Console.Write("กรุณาระบุยอดเงินที่ต้องการทอน: ");
int amount = int.Parse(Console.ReadLine()); // สมมติ 857

// 1. คำนวณแบงก์ 100
int bank100 = amount / 100;     // 857 / 100 ได้ 8 ใบ
amount = amount % 100;          // ยอดเงินที่เหลือจากแบงก์ร้อย: 857 % 100 = 57 บาท (อัปเดตตัวแปรเดิม)

// 2. คำนวณแบงก์ 50 จากยอดเงินที่เหลือ
int bank50 = amount / 50;       // 57 / 50 ได้ 1 ใบ
amount = amount % 50;           // ยอดเงินเหลือ: 57 % 50 = 7 บาท

// 3. คำนวณเหรียญ 10 จากยอดเงินที่เหลือ
int coin10 = amount / 10;       // 7 / 10 ได้ 0 เหรียญ
amount = amount % 10;           // ยอดเงินเหลือ: 7 % 10 = 7 บาท (ซึ่งจะถูกปัดทิ้งตามโจทย์)

// แสดงผลลัพธ์
Console.WriteLine("=== สรุปการทอนเงิน ===");
Console.WriteLine("แบงก์ 100: " + bank100 + " ใบ");
Console.WriteLine("แบงก์ 50: " + bank50 + " ใบ");
Console.WriteLine("เหรียญ 10: " + coin10 + " เหรียญ");
Console.WriteLine("(เศษที่ปัดทิ้ง: " + amount + " บาท)");
```

**โจทย์ที่ 2:** เดาผลลัพธ์ของโค้ดโดยไม่ต้องรัน
```csharp
int x = 5;
x += 3;
x *= 2;
x--;
Console.WriteLine(x);
```
**เฉลย:** ผลลัพธ์สุดท้ายคือ **`15`**
**การวิเคราะห์ทีละบรรทัด:**
1. `x = 5;` (x เริ่มที่ 5)
2. `x += 3;` (x = 5 + 3 กลายเป็น 8)
3. `x *= 2;` (x = 8 * 2 กลายเป็น 16)
4. `x--;` (x = 16 - 1 กลายเป็น 15)

## 🔥 เฉลย Challenge

**โจทย์:** สกัดตัวเลขหลักร้อยออกจาก `12345` โดยใช้แค่ `/` และ `%` (ห้ามแปลเป็นข้อความ)
**เฉลย:**
นี่คือเทคนิคทางคณิตศาสตร์ในการเลื่อนจุดและตัดเศษทิ้ง

```csharp
int n = 12345;

// ขั้นตอนที่ 1: เลื่อนจุดทศนิยม 2 ตำแหน่งด้วยการหาร 100
// เนื่องจาก n เป็น int การหารจะตัดเศษทิ้ง
// 12345 / 100 จะได้ผลลัพธ์คือ 123
int step1 = n / 100;

// ขั้นตอนที่ 2: ดึงเอาตัวเลขตัวสุดท้ายออกมา (ซึ่งก็คือหลักหน่วยของ 123)
// การทำ % 10 ของตัวเลขใดๆ จะเป็นการดึงตัวเลขตัวสุดท้ายออกมาเสมอ
// 123 % 10 จะได้ผลลัพธ์คือ 3
int hundredsDigit = step1 % 10;

Console.WriteLine("ตัวเลขหลักร้อยคือ: " + hundredsDigit);
```

**การเขียนแบบบรรทัดเดียวสั้นๆ:**
```csharp
int result = (n / 100) % 10; // ได้ 3
```
*เทคนิคนี้ทรงพลังมาก ใช้บ่อยมากในการเขียนโปรแกรมระดับฮาร์ดแวร์ เกมกราฟิก หรือการสกัดข้อมูลระดับ Bit!*
