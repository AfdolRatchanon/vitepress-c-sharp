# C# คืออะไร?

C# (ออกเสียงว่า "ซีชาร์ป") เป็นภาษาโปรแกรมเชิงวัตถุ (Object-Oriented Programming) ที่พัฒนาโดย Microsoft ซึ่งทำงานบน .NET framework หรือ .NET Core/.NET 5+

## ทำไมถึงควรเรียน C#
- **ใช้งานได้หลากหลาย**: สามารถพัฒนาได้ทั้ง Web, Desktop, Mobile (Xamarin/MAUI), เกม (Unity) และ Cloud/AI
- **มีโครงสร้างชัดเจน**: เป็นภาษาที่มี Type-safe แข็งแกร่ง ช่วยลดข้อผิดพลาดในการเขียนโปรแกรม
- **ชุมชนผู้ใช้งานขนาดใหญ่**: มีเอกสารอ้างอิงมากมายและหาความช่วยเหลือได้ง่าย
