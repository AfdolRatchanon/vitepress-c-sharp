# เฉลย: String Interpolation & Formatting (การปั้นและจัดรูปแบบ)

## 🎯 เฉลย Mini Exercise

### **โจทย์ที่ 1: เครื่องมือแต่งหน้าสถิติ (The Data Formatter)**
**เป้าหมาย:** สั่งให้ปีกกาของ String Interpolation แปลง `0.87654` เป็น `87.65%`

**เฉลย:**
หัวใจสำคัญคือรหัส `:P` (Percent) ซึ่งคอมพิวเตอร์จะแสนรู้มาก มันจะรู้ทันทีว่า "อ๋อ เปอร์เซ็นต์ แปลว่าต้องเอาไปคูณ 100 ก่อน แล้วค่อยเติมสัญลักษณ์ % ให้"
ส่วนเลข 2 ข้างหลัง P คือคำสั่งล็อกทศนิยม 2 ตำแหน่ง
```csharp
double successRate = 0.87654;

// ใช้ String Interpolation ($) และรหัสแต่งหน้า (:P2)
Console.WriteLine($"อัตราความสำเร็จในภารกิจนี้คือ: {successRate:P2}");
// ผลลัพธ์: อัตราความสำเร็จในภารกิจนี้คือ: 87.65%
```

### **โจทย์ที่ 2: ระบบล็อกอินปลอมตัว (The PadLeft Hacker)**
**เป้าหมาย:** รับเลขอะไรมาก็ตาม ต้องยัดเลข `0` ไว้ข้างหน้าให้ครบ 5 หลักเสมอ

**เฉลยและวิธีทำ:**
`.PadLeft(ความยาวทั้งหมด, 'ตัวอักษรที่จะเติม')` คือพระเอกของงานนี้
```csharp
using System;

class Program
{
    static void Main()
    {
        Console.Write("กรอกรหัสพนักงาน: ");
        string inputId = Console.ReadLine(); // สมมติว่าพิมพ์ 7

        // บังคับว่า "ความยาวรวมต้องเป็น 5" ถ้าไม่ถึง ให้เอา '0' อุดจากทางซ้ายให้เต็ม!
        string paddedId = inputId.PadLeft(5, '0');

        Console.WriteLine($"[ระบบรปภ.]: รหัสประจำตัวของคุณคือ {paddedId}");
        // ผลลัพธ์: [ระบบรปภ.]: รหัสประจำตัวของคุณคือ 00007
    }
}
```

## 🔥 เฉลย Challenge (มหกรรมบิลเงินสดอัจฉริยะ The Dynamic Invoice Builder)

**เป้าหมาย:** วาดตารางใบเสร็จด้วยลูป โดยชื่อสินค้าต้องชิดซ้าย (15 ช่อง) ราคาสินค้าต้องชิดขวา (10 ช่องพร้อมจุลภาคทศนิยม) และมียอดรวมตรงเป๊ะในบรรทัดสุดท้าย!

**กระบวนการคิดระดับสถาปนิก (Architectural Alignment):**
การสร้างตาราง Console ถือเป็นศิลปะแขนงหนึ่ง! เราต้องใช้สูตร **`{ตัวแปร, การผลักซ้ายขวา : การแต่งหน้าตัวเลข}`** 
- ชื่อสินค้า ชิดซ้ายกว้าง 15 -> `{product, -15}`
- ราคาสินค้า ชิดขวากว้าง 10 โชว์เงินทศนิยม -> `{price, 10:N2}`

```csharp
using System;

namespace InvoiceGenerator
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine("=== เครื่องจักรพิมพ์ใบเสร็จอัจฉริยะ (Smart POS) ===\n");

            // ฐานข้อมูลตะกร้าสินค้า
            string[] products = { "Apple", "Watermelon", "Gum" };
            double[] prices = { 15.5, 120.0, 5.0 };

            // ตัวแปรสะสมยอดเงินรวม
            double grandTotal = 0;

            // -----------------------------------------------------------
            // 1. วาดหัวตาราง (Header)
            // -----------------------------------------------------------
            // ITEM ชิดซ้าย 15 ช่อง | PRICE ชิดขวา 10 ช่อง
            Console.WriteLine($"{"ITEM", -15} | {"PRICE", 10}");
            Console.WriteLine(new string('-', 28)); // ขีดเส้นประยาว 28 ช่อง (15 + 3รอยต่อ + 10)

            // -----------------------------------------------------------
            // 2. วงลูปปั่นตาราง (Data Rendering)
            // -----------------------------------------------------------
            for (int i = 0; i < products.Length; i++)
            {
                // ดึงข้อมูลแต่ละแถวมาเตรียมไว้
                string currentProduct = products[i];
                double currentPrice = prices[i];

                // 🌟 ท่าไม้ตาย! ผสม Alignment เข้ากับ Formatting 🌟
                string rowLine = $"{currentProduct, -15} | {currentPrice, 10:N2}";
                Console.WriteLine(rowLine);

                // บวกเลขสะสมยอดรวมไปในตัว
                grandTotal += currentPrice;
            }

            // -----------------------------------------------------------
            // 3. วาดส่วนสรุปยอด (Footer)
            // -----------------------------------------------------------
            Console.WriteLine(new string('-', 28));
            
            // ใช้ "TOTAL" ผลักซ้าย 15 ช่อง เพื่อให้เส้นกั้น | ตรงกับบรรทัดบนเป๊ะ!
            Console.WriteLine($"{"TOTAL", -15} | {grandTotal, 10:N2}");
        }
    }
}
```

**ผลลัพธ์บนหน้าจอ (เป๊ะทุกมิลลิเมตร):**
```text
=== เครื่องจักรพิมพ์ใบเสร็จอัจฉริยะ (Smart POS) ===

ITEM            |      PRICE
----------------------------
Apple           |      15.50
Watermelon      |     120.00
Gum             |       5.00
----------------------------
TOTAL           |     140.50
```

**สิ่งที่น่าเรียนรู้ (UI Lesson):**
ถ้าคุณทำข้อนี้สำเร็จ คุณก็พร้อมสร้าง Console Menu หรือ Report Dashboard ที่สวยงามให้องค์กรใช้งานได้แล้วครับ!
