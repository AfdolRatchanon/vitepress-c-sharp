# เฉลย: Nested Loops (การซ้อนลูป)

## 🎯 เฉลย Mini Exercise

### **โจทย์ที่ 1: ตารางหมากรุก (Checkerboard Pattern)**
**เป้าหมาย:** พิมพ์ตาราง 5x5 สลับ X และ O อย่างสวยงาม

**กระบวนการคิดและเฉลย:**
การวาดลวดลายสลับฟันปลาในตารางแบบนี้ มีสูตรตายตัวระดับจักรวาลคือการนำเอาตัวแปรนับรอบ (Row กับ Col) มา "บวกกัน" แล้วหาว่าเป็น "เลขคู่ หรือ เลขคี่" (Modulo ด้วย 2)
- แถว 1 คอลัมน์ 1 `(1+1 = 2)` -> คู่ (พิมพ์ X)
- แถว 1 คอลัมน์ 2 `(1+2 = 3)` -> คี่ (พิมพ์ O)
- แถว 2 คอลัมน์ 1 `(2+1 = 3)` -> คี่ (พิมพ์ O)  <-- สังเกตว่ามันสลับกันพอดีเป๊ะแม้อยู่คนละบรรทัด!

```csharp
Console.WriteLine("=== กระดานหมากรุก (5x5 Checkerboard) ===");

// ลูปแกน Y (แถวแนวตั้ง 5 แถว)
for (int row = 1; row <= 5; row++)
{
    // ลูปแกน X (ช่องแนวนอน 5 ช่องในแต่ละแถว)
    for (int col = 1; col <= 5; col++)
    {
        // ตรวจสอบความสัมพันธ์ระหว่างพิกัด X และ Y
        if ((row + col) % 2 == 0)
        {
            Console.Write("X "); // พิมพ์ X โดยไม่ลงบรรทัดใหม่
        }
        else
        {
            Console.Write("O "); // พิมพ์ O
        }
    }
    
    // จบแกน X ทุกๆ 5 ช่อง ต้องสั่งให้ตัดขึ้นบรรทัดใหม่เสมอ
    Console.WriteLine();
}
```

### **โจทย์ที่ 2: พิระมิดดาวครึ่งซีก (Half-Pyramid Stars)**
**เป้าหมาย:** พิมพ์สามเหลี่ยมดาว ที่จำนวนดาวสอดคล้องกับเลขชั้น (ชั้น 1 ดาว 1, ชั้น 5 ดาว 5)

**เฉลยและวิเคราะห์กระบวนการ (Gear Mechanics):**
จุดหักเหที่ทำให้โจทย์ข้อนี้แตกต่างจากการวาดสี่เหลี่ยมคือ "ขอบเขตของลูปชั้นใน ไม่ได้คงที่อีกต่อไป แต่มันผูกติดกับค่าของลูปชั้นนอก"
ในขณะที่ลูปชั้นนอก (เข็มนาที) ขยับ มันได้ส่ง "โควต้าการหมุน" ให้ลูปชั้นใน (เข็มวินาที) เปลี่ยนแปลงตามไปด้วย

```csharp
Console.WriteLine("=== พิระมิดครึ่งซีก (Half-Pyramid) ===");

int maxLevels = 5;

// ลูปคุมจำนวนชั้น (ลงมาทีละบรรทัด 5 รอบ)
for (int row = 1; row <= maxLevels; row++)
{
    // ลูปวาดดาว (สำคัญมาก: เงื่อนไขคือ col <= row)
    // รอบที่ row=1: ลูปนี้ทำงานแค่ col=1 (ดาว 1 ดวง) แล้วจบลูปเลย
    // รอบที่ row=3: ลูปนี้ทำงาน col=1, 2, 3 (ดาว 3 ดวง) แล้วจึงจบ
    for (int col = 1; col <= row; col++)
    {
        Console.Write("*");
    }
    
    // จบการวาดดาวแต่ละชั้น ต้องเคาะขึ้นบรรทัดใหม่
    Console.WriteLine();
}
```

## 🔥 เฉลย Challenge (พิระมิดอียิปต์ The Perfect Egyptian Pyramid)

**โจทย์:** สร้างสามเหลี่ยมดาวแบบเต็มใบ สมมาตรซ้ายขวา
**เป้าหมายทางวิศวกรรม:** ฝึกการเชื่อมต่อลูป (Sequential Inner Loops) ไว้ภายในลูปห่อหุ้ม 1 ชั้น เพื่อวาดโครงสร้างที่มีช่องว่าง (Spaces) ผสมกับสัญลักษณ์ (Stars) ในสัดส่วนที่คำนวณด้วยคณิตศาสตร์

**กระบวนการคิดและเฉลยเชิงลึก (Algorithmic Architecture):**
ลองแยกโครงสร้างของรูปออกเป็นตาราง Excel ดู:
ชั้นที่ 1: ช่องว่าง 4 ช่อง, ดาว 1 ดวง
ชั้นที่ 2: ช่องว่าง 3 ช่อง, ดาว 3 ดวง
ชั้นที่ 3: ช่องว่าง 2 ช่อง, ดาว 5 ดวง
ชั้นที่ 4: ช่องว่าง 1 ช่อง, ดาว 7 ดวง
ชั้นที่ 5: ช่องว่าง 0 ช่อง, ดาว 9 ดวง

**สกัดสูตรคณิตศาสตร์ (Formulas):**
- โควต้าของ "ช่องว่าง" = `(จำนวนชั้นทั้งหมด) - ชั้นปัจจุบัน` (เช่น 5 - 1 = 4)
- โควต้าของ "ดวงดาว" = `(2 * ชั้นปัจจุบัน) - 1` (เช่น (2 * 2) - 1 = 3)

```csharp
Console.WriteLine("=================================");
Console.WriteLine(" 🐫 THE GREAT EGYPTIAN PYRAMID 🐫 ");
Console.WriteLine("=================================");

Console.Write("คุณต้องการสร้างพิระมิดความสูงกี่ชั้น? : ");
int totalRows = int.Parse(Console.ReadLine());

Console.WriteLine();

// 1. ลูปโครงข่ายหลัก คุมความสูงแนวตั้งทั้งหมด
for (int row = 1; row <= totalRows; row++)
{
    // ----------------------------------------------------
    // ลูปย่อยตัวที่ 1: หน้าที่วาด "บรรยากาศ (ช่องว่าง)" ผลักรูปทรงไปตรงกลาง
    // ----------------------------------------------------
    int spaceCount = totalRows - row;
    for (int s = 1; s <= spaceCount; s++)
    {
        Console.Write(" "); // พิมพ์เคาะเว้นวรรค 1 ที
    }

    // ----------------------------------------------------
    // ลูปย่อยตัวที่ 2: หน้าที่ก่อ "อิฐ (ดวงดาว)"
    // ----------------------------------------------------
    int starCount = (2 * row) - 1;
    for (int star = 1; star <= starCount; star++)
    {
        Console.Write("*"); // พิมพ์ดาวต่อจากช่องว่าง
    }

    // เมื่อแก๊งลูปย่อยทั้งสองตัว ทำหน้าที่ในชั้นนี้เสร็จสมบูรณ์
    // สั่งขึ้นบรรทัดใหม่ เพื่อเริ่มสร้างฐานชั้นต่อไป
    Console.WriteLine();
}
Console.WriteLine("\n[สร้างสิ่งก่อสร้างมหัศจรรย์เสร็จสมบูรณ์]");
```

**บทสรุปสำหรับยอดมนุษย์:**
ถ้าคุณสามารถเข้าใจทิศทางการไหลของโค้ดพิระมิดอียิปต์นี้ได้... ยินดีด้วยครับ! สมองของคุณได้ปรับโครงสร้างเพื่อรองรับ "มิติเชิงลอจิกซ้อนลอจิก" เป็นที่เรียบร้อย คุณมีความพร้อมเกินร้อยในการไปเผชิญหน้ากับ Array 2 มิติ หรือระบบประมวลผลพิกเซลของภาพถ่ายในขั้นสูงต่อไป!
