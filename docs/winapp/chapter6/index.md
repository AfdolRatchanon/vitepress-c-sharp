# บทที่ 6: Menus, Toolbars & Status

> 💡 **ภาพรวม:** บทนี้จะสอนสร้างระบบเมนูและ Toolbar แบบมืออาชีพเหมือนโปรแกรม Notepad หรือ Microsoft Word ด้วย MenuStrip, ToolStrip, StatusStrip และ ContextMenuStrip ซึ่งเป็นองค์ประกอบสำคัญของแอปพลิเคชันระดับโปรที่ขาดไม่ได้

## 🎯 สิ่งที่จะได้เรียนรู้ในบทนี้
- ✅ MenuStrip: สร้างเมนูหลักแบบ File > Edit > View
- ✅ ToolStrip: สร้างแถบเครื่องมือพร้อมไอคอน
- ✅ StatusStrip: แสดงข้อมูลสถานะด้านล่างหน้าจอ
- ✅ ContextMenuStrip: เมนูคลิกขวาสำหรับ Control ต่างๆ

## 🔗 Prerequisite (ความรู้ที่ต้องมีก่อน)
> ⚠️ นักเรียนควรเข้าใจสิ่งต่อไปนี้ก่อนเริ่มบทนี้:
- บทที่ 5: Layout & Containers (Panel, TabControl)
- การสร้าง Event Handler และการใช้ MessageBox
