# บทที่ 17: UI Customization & Graphics (GDI+)

> 💡 **ภาพรวม:** สอนการวาดกราฟิกบนฟอร์มด้วย GDI+ เช่น การวาดเส้น, รูปร่าง, การทำกราฟเส้นอย่างง่าย และการสร้าง User Interface ที่ดูทันสมัย (Modern UI) รวมถึงการทำแอปให้รองรับหลายภาษา (Localization)

## 🎯 สิ่งที่จะได้เรียนรู้ในบทนี้
- ✅ การวาดรูปทรงและกราฟิกด้วย `System.Drawing` และ `Graphics` (GDI+)
- ✅ การสร้าง Custom Control ของตัวเองเพื่อความสวยงาม
- ✅ การทำ Dashboard และกราฟแท่งแบบ Custom
- ✅ การปรับแต่ง UI ให้ดูทันสมัย (Flat Design)
- ✅ การรองรับหลายภาษา (Localization) ในแอปเดียว

## 🔗 Prerequisite (ความรู้ที่ต้องมีก่อน)
> ⚠️ นักเรียนควรเข้าใจสิ่งต่อไปนี้ก่อนเริ่มบทนี้:
- บทที่ 5: Event Handlers (คลิกเมาส์, เลื่อนเมาส์)
- บทที่ 10: Timer (สำหรับการทำ Animation อย่างง่าย)
