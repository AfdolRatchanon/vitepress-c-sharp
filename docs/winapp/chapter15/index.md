# บทที่ 15: Consuming Web APIs & JSON

> 💡 **ภาพรวม:** สอนการดึงข้อมูลจากอินเทอร์เน็ตด้วย HttpClient, แปลงข้อมูล JSON ด้วย System.Text.Json และการแสดงผลข้อมูล API บน WinForms แบบ Async เพื่อสร้างแอปพลิเคชันที่เชื่อมต่อโลกภายนอก

## 🎯 สิ่งที่จะได้เรียนรู้ในบทนี้
- ✅ ใช้ HttpClient ดึงข้อมูลจาก REST API
- ✅ Deserialize JSON เป็น C# Object อย่างง่าย
- ✅ แสดงข้อมูลจาก API ลงใน DataGridView หรือ ListBox
- ✅ จัดการ Network Error (เช่น ไม่มีเน็ต) อย่างเหมาะสม

## 🔗 Prerequisite (ความรู้ที่ต้องมีก่อน)
> ⚠️ นักเรียนควรเข้าใจสิ่งต่อไปนี้ก่อนเริ่มบทนี้:
- บทที่ 10: async/await (สำคัญมากเพราะการเรียก API ต้องเป็น Async)
- ความรู้เรื่อง Class และ List
