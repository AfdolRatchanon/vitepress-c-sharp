# บทที่ 5: Layout & Containers

> 💡 **ภาพรวม:** บทนี้จะสอนการจัดระเบียบ Control บนหน้าจอให้สวยงามและ Responsive ด้วย Container Controls เช่น Panel, GroupBox, TabControl และ SplitContainer ซึ่งเป็นหัวใจสำคัญของการสร้าง UI ที่ดูเป็นระเบียบและใช้งานง่าย

## 🎯 สิ่งที่จะได้เรียนรู้ในบทนี้
- ✅ Panel & GroupBox: จัดกลุ่ม Control ให้เป็นระเบียบ
- ✅ TabControl: สร้างฟอร์มแบบหลาย Tab คล้าย Settings ของ Windows
- ✅ SplitContainer & FlowLayoutPanel: จัดวาง Layout แบบยืดหยุ่น
