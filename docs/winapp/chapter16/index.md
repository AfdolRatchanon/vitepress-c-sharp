# บทที่ 16: NuGet Packages & Data Exporting

> 💡 **ภาพรวม:** สอนการใช้งาน NuGet Package Manager เพื่อติดตั้งไลบรารีเสริมจากนักพัฒนาทั่วโลก และประยุกต์ใช้งานด้วยการเขียนโค้ด Export ข้อมูลจากแอปพลิเคชันของเราออกเป็นไฟล์ Microsoft Excel (.xlsx) และไฟล์เอกสาร PDF

## 🎯 สิ่งที่จะได้เรียนรู้ในบทนี้
- ✅ รู้จักและใช้งาน NuGet Package Manager ใน Visual Studio
- ✅ ติดตั้งไลบรารีภายนอกเข้าสู่โปรเจกต์
- ✅ ส่งออกข้อมูลตาราง (DataGridView) เป็นไฟล์ Excel ด้วยไลบรารี `EPPlus` หรือ `ClosedXML`
- ✅ ส่งออกรายงานเป็นไฟล์ PDF ด้วยไลบรารี `iText7` หรือ `PdfSharp`

## 🔗 Prerequisite (ความรู้ที่ต้องมีก่อน)
> ⚠️ นักเรียนควรเข้าใจสิ่งต่อไปนี้ก่อนเริ่มบทนี้:
- บทที่ 11: File I/O (ความเข้าใจเรื่อง File Path และ Stream)
- บทที่ 13: Data Binding (การใช้งาน DataTable/DataGridView)
