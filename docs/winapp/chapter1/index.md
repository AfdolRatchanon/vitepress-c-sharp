# บทที่ 1: Introduction to Windows App & Visual Studio IDE

> 💡 **ภาพรวม:** บทนี้จะพานักเรียนก้าวข้ามจากการเขียนโปรแกรมบนหน้าจอสีดำ (Console) เข้าสู่โลกของการสร้างโปรแกรมที่มีหน้าต่าง (Windows Application) โดยใช้ Visual Studio IDE

## 🎯 สิ่งที่จะได้เรียนรู้ในบทนี้
- ✅ ความแตกต่างระหว่าง Console Application และ Windows Forms
- ✅ โครงสร้างโปรเจกต์ Windows Forms และไฟล์ที่สำคัญ (Form1.cs, Form1.Designer.cs)
- ✅ การใช้งานหน้าต่างเครื่องมือ (Toolbox, Properties Window, Solution Explorer)
- ✅ สถาปัตยกรรมการเขียนโปรแกรมแบบตอบสนองต่อเหตุการณ์ (Event-Driven Programming)

## 🔗 Prerequisite (ความรู้ที่ต้องมีก่อน)
> ⚠️ นักเรียนควรเข้าใจสิ่งต่อไปนี้ก่อนเริ่มบทนี้:
- โครงสร้างภาษา C# เบื้องต้น (ตัวแปร, if-else, วิธีการสร้างโปรเจกต์ Console) จากบทเรียน Console App
