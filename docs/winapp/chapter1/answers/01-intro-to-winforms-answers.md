# เฉลย 01 - รู้จักกับ Windows Forms และสร้างโปรเจกต์แรก

## 🎯 เฉลยโจทย์ฝึกปฏิบัติ (Mini Exercise)

**โจทย์ที่ 1:** เปลี่ยน Property `Text` ของ Form เป็นชื่อ-นามสกุล และเปลี่ยน `BackColor` เป็นสีชมพู
**วิธีทำ:** 
1. คลิกที่หน้าต่าง Form1
2. ไปที่หน้าต่าง Properties มุมขวาล่าง
3. ในช่อง **Text** พิมพ์ชื่อและนามสกุลของคุณ เช่น `Afdol Ratchanon`
4. ในช่อง **BackColor** คลิกที่ลูกศรชี้ลง เลื่อนไปที่แถบ Web หรือ Custom แล้วเลือกสีชมพู (Pink)
5. กดปุ่ม `F5` โปรแกรมจะแสดงหน้าต่างสีชมพูพร้อมชื่อของคุณบนแถบด้านบน

**โจทย์ที่ 2:** ปรับ Property `MaximizeBox` เป็น `False`
**วิธีทำ:** 
1. ในหน้าต่าง Properties ค้นหา `MaximizeBox`
2. เปลี่ยนค่าจาก `True` เป็น `False`
3. เมื่อกด `F5` สังเกตที่ปุ่มสี่เหลี่ยมสำหรับขยายหน้าต่างให้เต็มจอ (มุมขวาบน) จะกลายเป็นสีเทา (Disable) และไม่สามารถคลิกขยายได้

## 🔥 เฉลย Challenge
**โจทย์:** ค้นหา Property `Opacity` และเปลี่ยนค่าเป็น `50%`
**วิธีทำ:**
1. ในหน้าต่าง Properties ค้นหาคำว่า `Opacity`
2. เปลี่ยนตัวเลขจาก `100%` เป็น `50%`
3. เมื่อกดรัน (F5) โปรแกรมที่ปรากฏขึ้นจะมีความโปร่งแสง ทำให้มองทะลุเห็นพื้นหลังของเดสก์ท็อปได้ 50%
