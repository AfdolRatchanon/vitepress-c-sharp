# เฉลย 02 - เหตุการณ์ (Event) และโค้ดเบื้องหลัง (Code-Behind)

## 🎯 เฉลยโจทย์ฝึกปฏิบัติ (Mini Exercise)

**โจทย์ที่ 1:** ลากปุ่มมาเพิ่ม ตั้งชื่อ `btnExit` เปลี่ยนข้อความว่า "ออกจากโปรแกรม"
**วิธีทำ:** 
1. ลาก Button จาก Toolbox มาวางบน Form
2. ไปที่ Properties
3. ตั้ง `Name` เป็น `btnExit`
4. ตั้ง `Text` เป็น `ออกจากโปรแกรม`

**โจทย์ที่ 2:** สร้าง Event ให้ปุ่ม "ออกจากโปรแกรม" ปิดหน้าต่าง
**วิธีทำ:**
1. ดับเบิลคลิกที่ปุ่ม `btnExit`
2. ใส่คำสั่งปิดหน้าต่างลงในเมธอดที่สร้างขึ้นใหม่:

```csharp
private void btnExit_Click(object sender, EventArgs e)
{
    this.Close();
}
```

## 🔥 เฉลย Challenge
**โจทย์:** เพิ่มปุ่มชื่อ "โค้ดลับ!" แต่แสดง MessageBox เมื่อนำเมาส์ไปชี้ โดยไม่ต้องคลิก
**วิธีทำ:**
1. ลาก Button มาวางบน Form เปลี่ยน Text เป็น `โค้ดลับ!`
2. ตั้ง Name เป็น `btnSecret` (เป็นแนวปฏิบัติที่ดีแม้โจทย์จะไม่ได้สั่ง)
3. ในหน้าต่าง Properties ให้สังเกตไอคอนรูปสายฟ้า (⚡) ซึ่งเรียกว่าแท็บ **Events** ให้คลิกที่ไอคอนนั้น
4. เลื่อนหา Event ที่ชื่อว่า **`MouseEnter`**
5. ดับเบิลคลิกที่ช่องว่างด้านขวาของคำว่า `MouseEnter`
6. Visual Studio จะพามาที่ไฟล์ `.cs` อัตโนมัติ ให้พิมพ์โค้ดดังนี้:

```csharp
private void btnSecret_MouseEnter(object sender, EventArgs e)
{
    MessageBox.Show("คุณค้นพบความลับแล้ว!", "Secret Code");
}
```
7. เมื่อกดรัน (F5) ทันทีที่คุณเลื่อนเมาส์ไปทับปุ่มโดยยังไม่ทันกดคลิก กล่องข้อความก็จะเด้งขึ้นมาทันที!
