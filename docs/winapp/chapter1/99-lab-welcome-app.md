# 💻 Lab: Welcome Application (แอปพลิเคชันต้อนรับผู้ใช้)

> 💡 **เป้าหมาย:** สังเคราะห์ความรู้ทั้งหมดเกี่ยวกับการสร้าง Form, การลากวาง Control พื้นฐาน (Label, TextBox, Button), การตั้งค่า Properties ที่ดี และการเขียนโค้ดผูกกับ Event Click ออกมาเป็นแอปพลิเคชันแรกที่ใช้งานได้จริง

## 📖 ภาพรวมของโปรแกรม (Program Overview)

ใน Lab นี้ เราจะจำลองสถานการณ์ว่าเราเป็นนักพัฒนาซอฟต์แวร์ที่ต้องสร้างโปรแกรมทักทายพนักงานใหม่ของบริษัท โดยโปรแกรมจะมีกล่องรับข้อมูล (Input) ให้พนักงานกรอกชื่อของตนเอง และเมื่อกดปุ่มทักทาย โปรแกรมจะนำชื่อนั้นไปประมวลผลต่อกับข้อความต้อนรับ แล้วแสดงผลออกมาทางหน้าต่างแจ้งเตือนแบบ Pop-up (MessageBox)

นอกจากการทำงานหลักแล้ว โปรแกรมที่ดีควรมีฟังก์ชันอำนวยความสะดวกด้วย เราจึงจะเพิ่ม "ปุ่มล้างข้อมูล" เพื่อเคลียร์ช่องให้ว่าง และ "ปุ่มออกโปรแกรม" เพื่อความสมบูรณ์ของแอปพลิเคชัน

## 📝 ขั้นตอนการทำงาน (Step-by-Step)

### ขั้นตอนที่ 1: การออกแบบหน้าจอ (UI Design & Layout)
ในขั้นตอนนี้ เราจะโฟกัสแค่หน้าตาของโปรแกรม ยังไม่ต้องเขียนโค้ด C# ใดๆ ทั้งสิ้น

- [ ] **Task 1.1: ตั้งค่า Form หลัก**
  - สร้างโปรเจกต์ใหม่ชื่อ `LabWelcomeApp`
  - คลิกที่พื้นที่ว่างของ `Form1` แล้วเปลี่ยน `Text` เป็น `"Welcome App - ระบบต้อนรับพนักงาน"`
  - เปลี่ยน `StartPosition` เป็น `CenterScreen` เพื่อให้โปรแกรมเปิดตรงกลางจอเสมอ
  - เปลี่ยน `Font` ของ Form เป็นขนาด `12pt` (การเปลี่ยน Font ที่ Form จะทำให้ Control ทุกตัวที่ลากมาวางมีขนาดตัวอักษร 12pt ตามไปด้วยโดยอัตโนมัติ)

- [ ] **Task 1.2: ส่วนรับข้อมูล (Input)**
  - ลาก **Label** มาวางเหนือช่องกรอกข้อมูล เปลี่ยน Property `Text` เป็น `"กรุณากรอกชื่อของคุณ:"`
  - ลาก **TextBox** มาวาง (ใช้สำหรับให้ผู้ใช้พิมพ์ข้อความ) ปรับความกว้างให้เหมาะสม 
  - **สำคัญ:** เปลี่ยน Property `Name` ของ TextBox เป็น `txtName`

- [ ] **Task 1.3: ส่วนควบคุมการทำงาน (Buttons)**
  - ลาก **Button** มาวางตรงกลาง เปลี่ยน `Name` เป็น `btnSayHello` และเปลี่ยน `Text` เป็น `"ทักทาย!"`
  - ลาก **Button** มาวางมุมซ้ายล่าง เปลี่ยน `Name` เป็น `btnClear` และเปลี่ยน `Text` เป็น `"ล้างข้อมูล"`
  - ลาก **Button** มาวางมุมขวาล่าง เปลี่ยน `Name` เป็น `btnExit` และเปลี่ยน `Text` เป็น `"ออก"`

*(จุดเช็คพอยต์: ทดลองกด F5 รันโปรแกรมดูว่าหน้าตาแอปพลิเคชันแสดงผลถูกต้องและสวยงามหรือไม่ ปุ่มกดได้ไหม แต่กดแล้วจะยังไม่มีอะไรเกิดขึ้น)*

### ขั้นตอนที่ 2: การเขียนโค้ดเบื้องหลัง (Code-Behind & Logic)
ขั้นตอนนี้เราจะผูกจิตวิญญาณ (Logic) ให้กับหน้าตาโปรแกรม

- [ ] **Task 2.1: โค้ดปุ่มทักทาย (`btnSayHello`)**
  - ดับเบิลคลิกที่ปุ่ม `btnSayHello` เพื่อเปิดไฟล์ `Form1.cs`
  - สร้างตัวแปรประเภท `string` มารับค่าข้อความที่ผู้ใช้พิมพ์ โดยพิมพ์โค้ด `string userName = txtName.Text;`
  - ใช้คำสั่ง `MessageBox.Show()` เพื่อแสดงข้อความว่า "ยินดีต้อนรับคุณ [userName] เข้าสู่ระบบของเรา!"

- [ ] **Task 2.2: โค้ดปุ่มล้างข้อมูล (`btnClear`)**
  - กลับมาที่หน้า Design (Shift+F7) ดับเบิลคลิกที่ปุ่ม `btnClear`
  - การเคลียร์ช่องข้อความทำได้ 2 วิธี คือ `txtName.Text = "";` หรือใช้เมธอดสำเร็จรูป `txtName.Clear();` ให้นักเรียนเลือกใช้แบบใดแบบหนึ่ง
  - *(เทคนิคเพิ่มเติม: หลังจากกดล้างข้อมูลแล้ว เคอร์เซอร์เมาส์มักจะหายไป ให้นักเรียนเพิ่มคำสั่ง `txtName.Focus();` เพื่อสั่งให้เคอร์เซอร์กลับมากระพริบรอที่ช่อง TextBox อัตโนมัติ)*

- [ ] **Task 2.3: โค้ดปุ่มออกโปรแกรม (`btnExit`)**
  - ดับเบิลคลิกที่ปุ่ม `btnExit`
  - พิมพ์คำสั่งปิดฟอร์ม: `this.Close();`

*(จุดเช็คพอยต์: กด F5 ทดสอบระบบทั้งหมด พิมพ์ชื่อ -> กดทักทาย -> กดล้างข้อมูล -> พิมพ์ใหม่ -> กดออกโปรแกรม ต้องทำงานได้อย่างสมบูรณ์)*

## 🔥 Challenge (โจทย์ท้าทาย สำหรับคนที่ทำเสร็จก่อนเวลา!)

### Challenge 1: การจัดการข้อผิดพลาด (Validation)
ในชีวิตจริง ผู้ใช้อาจจะเผลอกดปุ่ม "ทักทาย" โดยที่ยังไม่ได้กรอกชื่อเลย ซึ่งถ้าเราไม่ป้องกันไว้ โปรแกรมก็จะพิมพ์ว่า "ยินดีต้อนรับคุณ เข้าสู่ระบบ" (เว้นว่างไว้) ซึ่งดูไม่เป็นมืออาชีพ

- **โจทย์:** ให้นำความรู้เรื่อง `if-else` จากวิชา Console มาเช็คว่า ถ้าตัวแปร `userName` ว่างเปล่า (เท่ากับ `""`) ให้โชว์ MessageBox แจ้งเตือนด่าผู้ใช้ว่า **"คุณยังไม่ได้กรอกชื่อเลย!"** พร้อมใส่ไอคอน `MessageBoxIcon.Warning` แทน
- **ถ้าใส่ข้อมูลถูกต้อง:** ค่อยแสดงข้อความท้อนรับปกติเหมือนที่ทำไป

### Challenge 2: พลังแห่งการกดปุ่ม Enter
คุณเคยไหมที่ขี้เกียจเอื้อมมือไปจับเมาส์เพื่อคลิกปุ่ม? เวลาคุณพิมพ์ชื่อเสร็จ คุณอยากกดปุ่ม **Enter** บนคีย์บอร์ด แล้วให้โปรแกรมทักทายทันทีเลย!

- **โจทย์:** ให้นักเรียนตั้งค่าให้ปุ่ม `btnSayHello` เปรียบเสมือนปุ่ม Default ของหน้าต่างนี้ 
- **วิธีทำ:** ให้ไปคลิกที่พื้นว่างๆ ของ `Form1` แล้วค้นหา Property ที่ชื่อว่า **`AcceptButton`** จากนั้นให้ตั้งค่าเลือกเป็นปุ่ม `btnSayHello` ลองรันโปรแกรม พิมพ์ชื่อ แล้วเคาะ Enter ดูสิ!

## 🗣️ สรุปสิ่งที่เรียนรู้จาก Lab นี้
1. ได้เรียนรู้ว่า Property `Text` ของ Form คือชื่อหน้าต่าง แต่ Property `Text` ของ TextBox คือข้อมูลที่อยู่ข้างในกล่อง
2. ได้รู้ว่าเราต้องแปลง Property `Text` ของ TextBox ให้อยู่ในรูปแบบตัวแปร `string` ก่อนนำไปใช้งานเสมอ
3. การทำ Validation เป็นสิ่งสำคัญมากในการเขียนโปรแกรม GUI
