# บทที่ 2: Essential Input & Output Controls

> 💡 **ภาพรวม:** บทนี้จะสอนใช้งาน Control พื้นฐานที่ขาดไม่ได้ในทุกแอปพลิเคชัน ได้แก่ Label, TextBox, Button และ MessageBox อย่างเจาะลึก รวมถึงการตรวจสอบข้อมูล (Input Validation) ก่อนนำไปประมวลผล

## 🎯 สิ่งที่จะได้เรียนรู้ในบทนี้
- ✅ การใช้ Label เพื่อแสดงข้อความและผลลัพธ์แบบ Dynamic
- ✅ การใช้ TextBox รับข้อมูลจากผู้ใช้และการกำหนด Properties ขั้นสูง
- ✅ เทคนิคการตรวจสอบข้อมูลก่อนนำไปประมวลผล (Input Validation)
- ✅ การแปลงชนิดข้อมูล (Type Conversion) จาก `string` เป็นตัวเลข

## 🔗 Prerequisite (ความรู้ที่ต้องมีก่อน)
> ⚠️ นักเรียนควรเข้าใจสิ่งต่อไปนี้ก่อนเริ่มบทนี้:
- บทที่ 1: การสร้างโปรเจกต์ Windows Forms และการสร้าง Event Handler ด้วยการดับเบิลคลิก
- ความรู้เรื่องตัวแปร ชนิดข้อมูล (`string`, `int`, `double`) และคำสั่ง `if-else` จาก C# Console
