# บทที่ 12: Database Integration with ADO.NET

> 💡 **ภาพรวม:** สอนการเชื่อมต่อ SQL Server ด้วย ADO.NET ด้วย SqlConnection, SqlCommand และ SqlDataReader พร้อมหลักการ Parameterized Query ป้องกัน SQL Injection

## 🎯 สิ่งที่จะได้เรียนรู้ในบทนี้
- ✅ เชื่อมต่อ SQL Server ด้วย SqlConnection และ Connection String
- ✅ ดึงข้อมูลด้วย SqlCommand และ SqlDataReader
- ✅ ใช้ Parameterized Query ป้องกัน SQL Injection
- ✅ จัดการ Connection อย่างปลอดภัยด้วย `using`

## 🔗 Prerequisite (ความรู้ที่ต้องมีก่อน)
> ⚠️ นักเรียนควรเข้าใจสิ่งต่อไปนี้ก่อนเริ่มบทนี้:
- SQL พื้นฐาน: SELECT, INSERT, UPDATE, DELETE, WHERE
- บทที่ 11: File I/O (ความคุ้นเคยกับการจัดการ Resource)
