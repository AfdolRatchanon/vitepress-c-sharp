# บทที่ 7: Dialog Windows

> 💡 **ภาพรวม:** บทนี้จะสอนการใช้งาน Common Dialogs ของ Windows ที่มีให้ใช้งานแบบสำเร็จรูป ได้แก่ OpenFileDialog, SaveFileDialog, ColorDialog, FontDialog และ FolderBrowserDialog ซึ่งช่วยให้ผู้ใช้สามารถโต้ตอบกับระบบไฟล์และการตั้งค่าได้อย่างมืออาชีพ

## 🎯 สิ่งที่จะได้เรียนรู้ในบทนี้
- ✅ OpenFileDialog & SaveFileDialog: เปิดและบันทึกไฟล์ผ่าน UI มาตรฐานของ Windows
- ✅ ColorDialog & FontDialog: เลือกสีและฟอนต์
- ✅ FolderBrowserDialog: เลือกโฟลเดอร์
- ✅ การสร้าง Custom Dialog Form ด้วยตัวเอง

## 🔗 Prerequisite (ความรู้ที่ต้องมีก่อน)
> ⚠️ นักเรียนควรเข้าใจสิ่งต่อไปนี้ก่อนเริ่มบทนี้:
- บทที่ 6: MenuStrip (มักใช้ Dialog ร่วมกับเมนู)
- ความรู้เรื่อง `System.IO` เบื้องต้น
