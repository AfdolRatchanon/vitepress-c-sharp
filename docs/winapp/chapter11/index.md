# บทที่ 11: File I/O & State Management

> 💡 **ภาพรวม:** สอนการอ่าน-เขียนไฟล์ Text/CSV ด้วย System.IO และการบันทึกการตั้งค่าผู้ใช้ด้วย Properties.Settings รวมถึง JSON Serialization ด้วย System.Text.Json

## 🎯 สิ่งที่จะได้เรียนรู้ในบทนี้
- ✅ File I/O: อ่าน-เขียนไฟล์ Text และ CSV
- ✅ Properties.Settings: บันทึก/โหลดการตั้งค่า User
- ✅ JSON Serialization: บันทึก Object ลงไฟล์ JSON
- ✅ การจัดการ Path และ Directory อย่างปลอดภัย
