# บทที่ 13: Data Binding & CRUD Operations

> 💡 **ภาพรวม:** สอนการดึงข้อมูลจาก Database มาแสดงใน DataGridView แบบ Data Binding และเขียนโค้ด CRUD ครบทุก Operation สำหรับสร้างระบบจัดการข้อมูลที่ใช้งานได้จริง

## 🎯 สิ่งที่จะได้เรียนรู้ในบทนี้
- ✅ ดึงข้อมูลจาก DB ลง DataGridView ด้วย DataAdapter & DataTable
- ✅ เพิ่มข้อมูล (INSERT) ผ่านฟอร์มกรอกข้อมูล
- ✅ แก้ไขข้อมูล (UPDATE) รายการที่เลือก
- ✅ ลบข้อมูล (DELETE) พร้อมยืนยัน

## 🔗 Prerequisite (ความรู้ที่ต้องมีก่อน)
> ⚠️ นักเรียนควรเข้าใจสิ่งต่อไปนี้ก่อนเริ่มบทนี้:
- บทที่ 12: ADO.NET Connection และ SqlCommand
