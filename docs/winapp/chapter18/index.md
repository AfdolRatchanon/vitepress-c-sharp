# บทที่ 18: Final Project, Git & Deployment

> 💡 **ภาพรวม:** เตรียมความพร้อมก่อนจบหลักสูตรด้วยการนำความรู้ทั้งหมดมาสร้างระบบขายหน้าร้าน (POS) สเกลจริง, เรียนรู้การใช้ระบบควบคุมเวอร์ชัน (Git), และการทำตัวติดตั้งโปรแกรม (Setup.exe) เพื่อนำไปติดตั้งที่เครื่องอื่น

## 🎯 สิ่งที่จะได้เรียนรู้ในบทนี้
- ✅ การออกแบบโครงสร้างโปรเจกต์ขนาดใหญ่
- ✅ การใช้ Git พื้นฐานสำหรับจัดการ Source Code
- ✅ การ Publish แอพพลิเคชันและสร้างไฟล์ Setup (.exe) สำหรับแจกจ่าย
- ✅ การบูรณาการความรู้ทั้งหมด (Database, API, OOP, GDI+, PDF)
