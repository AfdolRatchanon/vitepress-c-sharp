# 💻 Lab: พัฒนาระบบ POS (Point of Sale) ครบวงจร

> 💡 **เป้าหมาย:** สังเคราะห์ความรู้ทั้งหมด 18 บทเรียนเพื่อสร้างโปรแกรมร้านค้าปลีก (POS) แบบครบวงจรที่ใช้งานได้จริงในระดับ Production

## 📖 ภาพรวมของโปรเจกต์สุดท้าย

**ฟีเจอร์ที่ต้องมี:**
1. ระบบ Login และแบ่งสิทธิ์ (Admin / Cashier)
2. หน้าจอขายสินค้า (แสดงรูปสินค้า, แสกนบาร์โค้ด, ตะกร้าสินค้า, คำนวณเงินทอน)
3. ระบบพิมพ์ใบเสร็จรับเงิน (Export PDF)
4. ระบบจัดการสต๊อกหลังร้าน (CRUD สินค้าและหมวดหมู่)
5. หน้าจอ Dashboard สรุปยอดขายรายวัน (วาดด้วย GDI+)
6. เชื่อมต่อฐานข้อมูล SQL Server
7. รองรับภาษาไทย/อังกฤษ

## ⏱️ เวลาที่ใช้: 3-6 ชั่วโมง (Final Project)

## 📝 ข้อกำหนดของการพัฒนา (Project Requirements)

โปรเจกต์นี้จะไม่มีการให้โค้ดสำเร็จรูปทั้งหมด แต่จะเป็น Checklist เพื่อให้นักเรียนนำความรู้ที่เรียนมาลงมือทำด้วยตนเอง:

### Phase 1: Database Design (SQL Server)
สร้างตารางต่อไปนี้:
- `Users` (Id, Username, PasswordHash, Role)
- `Categories` (Id, Name)
- `Products` (Id, Name, Price, Stock, CategoryId, ImagePath)
- `Orders` (Id, OrderDate, TotalAmount, CashierId)
- `OrderDetails` (Id, OrderId, ProductId, Qty, UnitPrice)

### Phase 2: Project Architecture
จัดโครงสร้างโฟลเดอร์ดังนี้:
- `\Models` (User, Product, Order, OrderDetail)
- `\Repositories` (เชื่อม ADO.NET)
- `\Forms` (LoginForm, MainForm, CheckoutForm, DashboardForm)
- `\Helpers` (PdfHelper, HashHelper)

### Phase 3: Core Implementation
1. สร้าง **LoginForm** เมื่อ Login เสร็จให้สร้าง `AppSession` เพื่อเก็บ Role
2. สร้าง **MainForm** แบ่งจอเป็น 2 ฝั่ง ฝั่งขวาแสดงปุ่มกดสินค้า ฝั่งซ้ายแสดง ListView/DataGridView สำหรับตะกร้าสินค้า
3. การเพิ่มสินค้าลงตะกร้า: หากกดสินค้าที่เคยมีในตะกร้าแล้ว ให้บวกจำนวน (Qty) เพิ่มแทนการเพิ่มแถวใหม่
4. การคิดเงิน: สร้าง **CheckoutForm** ให้กรอกเงินรับมา คำนวณเงินทอน และกดปุ่ม "ยืนยัน"
5. เมื่อยืนยัน ให้บันทึกข้อมูลลงตาราง `Orders` และ `OrderDetails` (ใช้ SQL Transaction) พร้อมตัด Stock ในตาราง `Products`

### Phase 4: Advanced Features
1. **Export PDF:** สร้างปุ่มในตะกร้าให้พิมพ์ใบเสร็จรับเงิน (ใช้ iText7)
2. **Admin Dashboard:** สร้างหน้าจอสรุปยอดขายด้วย GDI+ กราฟแท่ง (ดูได้เฉพาะ Admin)
3. **Deployment:** เมื่อเสร็จแล้ว ทำการ Publish เป็นไฟล์ `Setup.exe`

## 🎉 บทสรุปหลักสูตร (Conclusion)

ขอแสดงความยินดี! หากคุณทำโปรเจกต์นี้สำเร็จ คุณได้ผ่านการเดินทางจากผู้ที่เขียนโค้ดเริ่มต้น จนสามารถสร้าง Application ระดับ Enterprise ที่มี UI ทันสมัย ระบบรักษาความปลอดภัย และการต่อฐานข้อมูลแบบมืออาชีพได้สำเร็จ 

**เส้นทางต่อไปของคุณ:**
- ศึกษา **Entity Framework Core (EF Core)** แทนการเขียน SQL ตรงๆ ด้วย ADO.NET
- ศึกษาสถาปัตยกรรม **MVVM** และก้าวไปสู่การทำแอปด้วย **WPF** หรือ **.NET MAUI**
- ศึกษาการทำ Web API (ASP.NET Core) เพื่อเปลี่ยนแอป Windows ให้ต่อเน็ตและดึงข้อมูลจาก Cloud ได้
