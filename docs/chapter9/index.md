# บทที่ 9: Object-Oriented Programming (ปฐมบทการเขียนโปรแกรมเชิงวัตถุ)

> 💡 **ภาพรวม:** ยินดีต้อนรับเข้าสู่ "แกนกลาง" ของภาษา C#! ตั้งแต่บทนี้เป็นต้นไป เราจะเลิกเขียนโค้ดแบบบนลงล่างยาวเหยียด (Procedural) และเข้าสู่โลกที่ทุกสิ่งทุกอย่างคือ "วัตถุ (Object)" ที่มีชีวิต มีข้อมูลเป็นของตัวเอง และสามารถสื่อสารกันได้ นี่คือกระบวนทัศน์ (Paradigm) ที่บริษัทซอฟต์แวร์ทั่วโลกใช้ในการสร้างแอปพลิเคชันขนาดใหญ่!

## 🎯 สิ่งที่จะได้เรียนรู้ในบทนี้

- ✅ **OOP Concepts:** ปรับจูนสมองใหม่ ทำความเข้าใจว่าสถาปัตยกรรมเชิงวัตถุคืออะไร และ 4 เสาหลักของมันคืออะไร
- ✅ **Classes & Objects:** การสร้าง "พิมพ์เขียว (Blueprint)" และเสกมันให้กลายเป็น "วัตถุที่มีชีวิต (Instance)" ในหน่วยความจำ Heap
- ✅ **Access Modifiers:** การตั้งยามรักษาความปลอดภัยให้ข้อมูล (`public` vs `private`)
- ✅ **Properties (Get/Set):** นวัตกรรมสุดล้ำของ C# ที่ทำหน้าที่เป็น "ด่านตรวจคนเข้าเมือง" คอยสกัดกั้นข้อมูลขยะไม่ให้ทำลายวัตถุของเรา
- ✅ **Constructors:** พิธีกรรมแรกเกิดของออบเจกต์ (การตั้งค่าเริ่มต้นก่อนนำไปใช้งาน)
- ✅ **Static Keyword:** พลังแห่งความตายตัว! ตัวแปรและฟังก์ชันที่แชร์ใช้ร่วมกันทั้งจักรวาลโดยไม่ต้อง `new`

## 🔗 Prerequisite (ความรู้ที่ต้องมีก่อน)

> ⚠️ ก่อนจะก้าวเข้าสู่โลกของ Class คุณต้องแม่นยำในอาวุธเหล่านี้:
- **Struct vs Class (บทที่ 8):** คุณต้องรู้แล้วว่า Class คือ "ตั๋วฝากของ (Reference Type)" ที่ชี้ไปหาคฤหาสน์บน Heap (ถ้ายังไม่แม่น ให้กลับไปทบทวนบท 8 ก่อน!)
- **Methods (บทที่ 5):** พฤติกรรมของ Object ทั้งหมด จะถูกเขียนอยู่ในรูปของ Method

## 🚀 ทำไมเราต้องเขียนโค้ดแบบ OOP?

ลองนึกภาพคุณต้องเขียนโปรแกรมจำลอง "ธนาคาร" ที่มีลูกค้า 1 ล้านคน
- **ยุคเก่า (Procedural):** คุณต้องสร้าง Array เก็บชื่อ 1 ล้านช่อง, Array เก็บยอดเงิน 1 ล้านช่อง แล้วมีฟังก์ชัน 1 ตัวชื่อ `Withdraw(accountIndex, amount)` โค้ดจะพันกันอีรุงตุงนัง และถ้าใครเผลอไปแก้ Array ยอดเงินมั่วๆ ธนาคารจะเจ๊งทันที!
- **ยุค OOP:** คุณสร้างกล่อง `BankAccount` ขึ้นมา 1 กล่อง ข้างในมีทั้ง "ยอดเงิน (Data)" และมีท่า "ถอนเงิน (Method)" ฝังตัวอยู่ด้วยกัน... ลูกค้า 1 ล้านคน ก็คือกล่อง 1 ล้านใบที่เป็นเอกเทศต่อกัน ใครจะถอนเงิน ก็ต้องเดินไปเคาะประตูเรียกกล่องของตัวเองเท่านั้น ไม่มีใครข้ามเส้นไปล้วงกระเป๋าคนอื่นได้!
