### **โจทย์ที่ 1: เพชฌฆาต IndexOutOfRange**
**โค้ดเจ้าปัญหา:**
```csharp
int[] numbers = { 10, 20, 30, 40, 50 };

for (int i = 0; i <= numbers.Length; i++)
{
    Console.WriteLine(numbers[i]);
}
```

**การวิเคราะห์วิกฤติ (The Off-By-One Error):**
นี่คือบั๊กยอดฮิตอันดับ 1 ของโลกในการเขียนโปรแกรมตระกูล C!
- `numbers.Length` มีค่าเท่ากับ **5** (เพราะมี 5 ตัว)
- ลูปถูกตั้งเงื่อนไขไว้ว่าให้ทำตราบใดที่ `i <= 5`
- คอมพิวเตอร์จะวิ่งพิมพ์ค่าตอนที่ `i=0`, `1`, `2`, `3`, `4` อย่างสวยงาม (พิมพ์ 10 ถึง 50 ออกมา)
- แต่ลูปยังไม่จบ! มันหมุนต่อให้ `i=5` (เพราะ 5 <= 5 ถือว่าเป็นจริง)
- แล้วมันก็สั่ง `Console.WriteLine(numbers[5]);` ... **💥 บู้มมมมมม!** 
- โปรแกรมแครชทันที เพราะช่องเบอร์ 5 ไม่มีตัวตนอยู่จริง (อาร์เรย์จบแค่เบอร์ 4) นี่แหละความโหดร้ายของโลก Zero-Based Indexing

**วิธีแก้ (Fix):**
เปลี่ยน `<= numbers.Length` ให้เป็น `< numbers.Length` ซะ! (เอาเครื่องหมายเท่ากับออก) พอเป็น `< 5` ลูปมันจะไปตายที่เลข 4 แล้วหยุดทันที

### **โจทย์ที่ 2: ระบบตรวจหาสระภาษาอังกฤษ (The Vowel Hunter)**
**เป้าหมาย:** สร้างอาร์เรย์เก็บสระ แล้วไล่ตรวจจับตัวอักษรในข้อความ

**เฉลยและวิธีทำ:**
การมอง `string` เป็น Array ของตัวอักษร (char) คือกุญแจสำคัญ

```csharp
using System;

class Program
{
    static void Main()
    {
        // 1. สร้าง Array ของสระ
        char[] vowels = { 'a', 'e', 'i', 'o', 'u' };

        Console.Write("กรุณาป้อนข้อความภาษาอังกฤษ: ");
        string input = Console.ReadLine().ToLower(); // แปลงเป็นพิมพ์เล็กให้หมดก่อน จะได้หาง่าย

        int vowelCount = 0;

        // 2. ลูปชั้นนอก: กวาดตัวอักษร "ทุกตัว" ในข้อความของผู้ใช้ทีละตัว
        foreach (char letter in input)
        {
            // 3. ลูปชั้นใน: เช็คว่าตัวอักษรปัจจุบัน มันไปตรงกับใครในอาร์เรย์สระไหม?
            bool isVowel = false;
            foreach (char v in vowels)
            {
                if (letter == v)
                {
                    isVowel = true;
                    break; // เจอว่าเป็นสระแล้ว ไม่ต้องหาสระตัวอื่นต่อ ให้หยุดลูปชั้นในทันที
                }
            }

            // ถ้าเจอว่าเป็นสระ ให้บวกแต้ม
            if (isVowel)
            {
                vowelCount++;
            }
        }

        Console.WriteLine($"\n[รายงาน] ข้อความนี้มีสระภาษาอังกฤษทั้งหมด {vowelCount} ตัว");
    }
}

// 💡 ทริคระดับโปร: ถ้ารู้จักเครื่องมือเสริม (Array.IndexOf) โค้ดด้านบนจะถูกยุบเหลือแค่นี้!

//     if (Array.IndexOf(vowels, letter) >= 0) { vowelCount++; }
```

## 🔥 เฉลย Challenge (การสลับร่างข้อมูลโดยไม่ง้อไลบรารี The Pure Array Reversal)

**เป้าหมาย:** สลับหน้าหลังข้อมูลในอาร์เรย์ โดยใช้แค่ `for` loop ล้วนๆ ห้ามใช้ `Array.Reverse` (เพื่อฝึกวิชา Algorithm)

**กระบวนการคิดระดับสถาปนิก (The Two-Pointer Logic):**
ถ้าเราสลับทุกช่องตั้งแต่ต้นจนจบ มันจะสลับกลับมาที่เดิม (ไร้ค่า) 
ดังนั้น เราต้องสลับมันจาก **"ซ้ายสุด ข้ามไปชน ขวาสุด"** แล้วค่อยๆ หุบเข้ามาตรงกลาง! 
จุดหยุดคือ **"กึ่งกลางอาร์เรย์ (Length / 2)"**

```csharp
using System;

namespace PureArrayReversal
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine("=== สุดยอดวิชาสลับร่างข้อมูล (Algorithm 101) ===\n");

            int[] data = { 1, 2, 3, 4, 5, 6, 7 }; // มี 7 ตัว (กึ่งกลางคือ 7/2 = 3)

            Console.WriteLine("[ก่อนสลับ]");
            foreach (int n in data) Console.Write(n + " ");
            Console.WriteLine("\n");

            // =======================================================
            // เครื่องยนต์ Algorithm (หุบจากนอกเข้าใน)
            // =======================================================
            int length = data.Length;

            // วิ่งแค่ "ครึ่งเดียว" พอ! (i < 3)
            for (int i = 0; i < length / 2; i++)
            {
                // หาระยะกระจัดของเป้าหมายฝั่งตรงข้าม 
                // ถ้า i=0 เป้าหมายฝั่งขวาคือ (7-1) - 0 = 6
                // ถ้า i=1 เป้าหมายฝั่งขวาคือ (7-1) - 1 = 5
                int oppositeIndex = (length - 1) - i;

                // ใช้วิชา "แก้วน้ำใบที่ 3 (Temp)" มาสลับที่กัน!
                int tempBox = data[i];                     // ดูดของฝั่งซ้ายมาเก็บไว้
                data[i] = data[oppositeIndex];             // เอาของฝั่งขวามาเททับใส่ฝั่งซ้าย
                data[oppositeIndex] = tempBox;             // เอาของที่เก็บไว้ เทใส่กลับฝั่งขวา
                
                Console.WriteLine($"[ระบบ] สลับตำแหน่ง {i} กับ {oppositeIndex}");
            }

            Console.WriteLine("\n[หลังสลับสำเร็จ!]");
            foreach (int n in data) Console.Write(n + " ");
            Console.WriteLine();
        }
    }
}
```

**ความรู้เสริมทางวิศวกรรม:**
Algorithm ตัวนี้คือสิ่งเดียวกับที่ฝังอยู่ในเมธอด `Array.Reverse()` ของ C# นั่นแหละครับ! การที่เราสามารถเขียนโค้ดเลียนแบบไลบรารีของไมโครซอฟต์ได้ ถือเป็นการปลดล็อคทักษะ Data Structures เบื้องต้นอย่างสมบูรณ์แบบ เพราะวิชาตรรกะแบบนี้ สามารถเอาไปประยุกต์เขียนโปรแกรมได้ในทุกภาษาบนโลก ไม่ว่าจะ Java, Python หรือ C++ ครับ!
